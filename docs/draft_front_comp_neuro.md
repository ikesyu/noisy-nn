# ノイズ場は神経修飾場である — 共有重み上に多重化された方策を「アドレスする」機能的資源としてのノイズ

## 1. 本稿の位置づけ（一段落）

Noise-modulated Neural Network（NNN）では、ユニットごとのノイズ強度 $\sigma_k$ がそのユニットの機能参加を決める。本稿はこの物理量を**神経修飾場（neuromodulatory field）のモデル**として提示し、「単一の共有重み集合の上に多重化して蓄えられた複数の方策を、ノイズ場が**アドレスする**」という機構仮説を立て、最小の実証（3 行動モードの動員デモ）と、その機構が**単なる鍵ではなく最適強度をもつ機能的資源**であることを示す確率共鳴（stochastic resonance, SR）の逆 U 字曲線を与える。

---

## 2. 中心的主張（この一文を守る）

> **神経修飾物質様のノイズ場は、単一の重み集合に多重化された複数の方策を選択的にアドレスする。ノイズはシンボリックな「鍵（look-up key）」ではなく、最適強度をもつ機能的資源（確率共鳴）である。**

**主張しないこと（重要な戦略判断）：**

- **行動の創発は主張しない。** デモの 3 行動（Foraging / Avoidance / Sheltering）は教師付きベクトル場回帰で設計的に与えたものであり、「創発した」と述べた瞬間に「行動は設計されている」という批判が正当になる。創発は今後の課題として明示的に繰り延べる。
- **神経修飾物質の生理学的モデルであるとは主張しない。** 主張するのは「ゲイン／興奮性の修飾によって機能サブネットが選択される」という計算原理の対応であり、特定の伝達物質の薬理学的再現ではない。

この 2 つを先に手放すことで、想定される 2 大批判（§6）が安価に回避でき、しかも残る主張（アドレッシング + SR）は現有の実験だけで支えられる。

---

## 3. ロジックの骨格（4 段）

論文の論理は「ノイズ場が方策を選ぶ」→「その選択は鍵ではなく資源である」→「ゆえに神経修飾の計算モデルとして意味がある」という順に閉じる。

| 段 | 主張 | 根拠 | 状態 |
|---|---|---|---|
| **L1. アドレッシング** | 重みを一切変えずに、ノイズ場だけで異なる行動が選択される | 行動デモ：1 つの共有重み集合 + 3 つのノイズ場 → 3 つの識別可能な行動ベクトル場 | **実装済み** |
| **L2. 多重化（分割ではない）** | 各行動は互いに重なりうるサブネットに蓄えられており、排他的な区画への分割ではない | サブネット重なり（Jaccard）と共有ユニットの損傷実験 | **未実施（§7-b）** |
| **L3. 資源性（鍵ではない）** | 動員されたサブネットの性能はノイズ強度に対して**内点最適**を持つ。鍵に最適強度は存在しない | SR 曲線：`--sweep train --model sample` で内点最適 $\sigma^*\approx0.8$–$1.2$、低 $\sigma$ 側で崩壊 | **実装済み・主要結果** |
| **L4. 連続制御** | ノイズ場は離散スイッチではなく連続場であり、場の補間が未学習の中間行動を生む | 場の補間（デモの `cycle` モード）を定量化 | **部分的（§7-c）** |

**なぜ L3 が本稿の科学的な芯なのか。** L1 だけでは「ノイズ場は one-hot なスイッチ（= 記号的な鍵）に過ぎない」という解釈を排除できない。鍵ならば、閾値を超える限りどんな強度でも同じ行動が選ばれるはずで、**最適強度は存在しない**。したがって内点最適の存在は、この対抗仮説に対する直接の反証になる。ノイズは「どれを選ぶか」を運ぶだけでなく、「選ばれた計算を成立させる」ために必要な量として振る舞っている。

## 4. 機構と平均場の区別（本稿の技術的な要点）

SR 曲線は、モデルの選び方によって結論が変わる。この区別自体が結果である。

- **`--model sample`（実ノイズ注入 + 交差閾値 $h>0$）＝ 機構**：閾値下の信号はノイズの助けなしには伝達されないため、低ノイズ側に**真の障壁**がある。学習水準スイープ（各 $\sigma$ で新規ネットを学習し、その $\sigma$ で採点）でも**内点最適**が残る。
- **`--model analytic`（交差の期待値・閾値なし）＝ 平均場**：低 $\sigma$ でも重みの再スケールで学習が成立するため、学習水準スイープでは**低 $\sigma$ 端が最適**になる。SR の障壁を持たない。

すなわち、**SR は平均場近似では消え、サンプルレベルの機構でのみ現れる**。論文の厳密な SR 図は `--sweep train --model sample` であり、`analytic` はその対照として並べる（`recipe_sr.md` §3 の実測済み区別）。テスト時スイープ（1 ネットで学習水準を固定し、テスト時のノイズだけを掃引）は全モデルで逆 U 字を示すが、「学習水準の近傍で誤差が最小になるのは構造上当然」という交絡があるため、補助図に留める。

**実測例（seed 7、`--sweep train --model sample`）**：刺激依存信号のピークは $\sigma_{\text{train}}=1.19$（内点）、端点 0.063 / 0.790 に対しピーク 0.803。タスク誤差は端点 0.414（低 $\sigma$ = 閾値下の崩壊）/ 0.030 に対し最小 0.014。最適点での場の分離度 1.430。※単一 seed の例であり、投稿には複数 seed の mean±std（`recipe_sr.md` R4 / Fig 5）が要る。

## 5. 生物学的な対応づけ（Introduction の設計）

Introduction は「ノイズ場 ↔ 神経修飾」の対応を、比喩ではなく**計算的性質の一致**として立てる。押さえるべき対応と、引用で埋めるべき箇所：

| NNN の性質 | 神経修飾の対応物 | 引用で補強すべき点 |
|---|---|---|
| ユニットごとのノイズ強度 $\sigma_k$ が参加を決める | ゲイン変調・興奮性変調による回路の実効的な参加 | 神経修飾のゲイン変調としての定式化 |
| 空間的に広がるノイズ場（バンプ状の動員パターン） | 拡散性・容積伝達（volume transmission）による広域かつ勾配的な放出 | 修飾は点対点ではなく場である |
| 場が方策を選ぶ（重みは不変） | 文脈依存計算・状態依存の回路再構成 | context-dependent computation / gain modulation |
| 最適強度の存在（SR） | 神経系における確率共鳴と、修飾物質濃度の逆 U 字（Yerkes–Dodson 的な用量応答） | SR の神経科学的先行研究、NNN の SR 基盤に関する既刊 2 本（Neurocomputing） |
| 場の選択が感覚・内的状態に駆動される | 修飾物質放出のセンサリ／内部状態によるゲーティング | デモの閉ループ規則が対応（放出の制御は本稿の主張範囲外） |

**位置づけの表明**：本稿は「修飾物質の生理を再現するモデル」ではなく、「**修飾がなぜ機能サブネットの選択として働きうるか**の計算原理の最小モデル」である。関連文献の中では、gating / gain modulation / context-dependent computation / stochastic resonance の交差点に置く。特に Masse et al.（PNAS 2018）の context-dependent gating は最近傍だが、あちらはランダム割当のゲート + 別途のシナプス安定化という二機構であり、本稿は**単一の物理量が選択と計算の両方を担う**点で異なる（この対比は `docs/idea_consolidation.md` §14.4 に詳しい）。

## 6. 想定される批判と応答

| 批判 | 応答 | 必要な結果 |
|---|---|---|
| **(a) 行動は設計されたものであり、創発していない** | そのとおりであり、創発は主張しない。主張は「設計された複数方策が単一重み集合に多重化され、ノイズ場でアドレスされる」ことにある | なし（主張の限定で回避） |
| **(b) ノイズ場は単なる one-hot の鍵ではないか** | 鍵に最適強度は存在しない。内点最適（SR）の存在が直接の反証 | **L3（済）**、複数 seed 版 |
| **(c) 単に区画分割されたサブネットのルックアップでは** | 場は重なりを持ち、共有ユニットの損傷は複数行動を同時に劣化させる（区画分割なら独立に壊れるはず） | **L2（未実施）** |
| **(d) 神経修飾のモデルとしてのスコープ適合性** | 記事種別を Hypothesis & Theory とし、§5 の対応表と引用で計算原理レベルの主張であることを明示 | Introduction の文献整備 |
| **(e) 決定論モデルでも同じことが言えるのでは** | 平均場（analytic）では SR が消え、低 $\sigma$ 端最適になる（§4）。閾値つきサンプル機構でのみ内点最適が現れる | **済**（analytic 対照） |

## 7. 残作業（投稿までに埋める）

- **(a) SR 図の完成**：高 $\sigma$ 側の低下が緩いため掃引を広げる（$\sigma\sim4$–5 まで）。複数 seed（R4）で内点最適と低 $\sigma$ 崩壊の seed 非依存性を示す。任意で analytic と $\mathbb E[\text{sample}]$ の重ね描き。
- **(b) 多重化の実証（L2）**：3 つの場の動員ユニット集合の重なり（Jaccard）を測り、**共有ユニットの損傷が複数行動を同時に劣化させる**ことを示す。区画分割仮説への直接の反証で、批判 (c) に対応する。現状未実装。
- **(c) 連続制御の定量化（L4）**：デモの `cycle` モードは場の補間を動画で見せるが、定量化されていない。補間した場が生む行動場を、対応する補間目標に対して評価し、**未学習の中間行動が滑らかに現れる**ことを曲線で示す。
- **(d) 図の整理**：`recipe_sr.md` §6 の Fig 1–5 の枠組みをそのまま使える。本文図は Fig 1（厳密 SR）・Fig 2（機構 vs 平均場）・行動デモのパネル図（3 場 × ベクトル場）・(b) の損傷図の 4 枚程度に絞る。
- **(e) 実装の配置**：スクリプトを `examples/` に戻すか、`recipe_sr.md` の参照を `tmp/` に合わせるかを決める（投稿時にコード公開するなら前者）。

## 8. 想定する構成

1. **Introduction** — 神経修飾のゲイン変調・容積伝達 → 「修飾は場である」→ 計算モデルとしての問い：場は何を選び、なぜノイズでなければならないのか。
2. **Model** — NNN の交差活性、ノイズ場、動員。Analytic（平均場）と Sample（機構）の関係（$\text{Analytic}\approx\mathbb E[\text{Sample}]$ を示すこと自体が小さな結果）。
3. **Result 1: Addressing** — 1 重み集合 + 3 場 → 3 行動（パネル図）。閉ループデモは補足動画へ。
4. **Result 2: Multiplexing** — 重なり（Jaccard）+ 共有ユニット損傷（§7-b）。
5. **Result 3: Stochastic resonance** — 学習水準スイープの内点最適、低 $\sigma$ 崩壊、機構 vs 平均場の対照。**本稿の主要結果**。
6. **Result 4: Continuous control**（任意）— 場の補間による未学習の中間行動。
7. **Discussion** — 神経修飾との対応（§5 の表）、gating / SR 文献との位置づけ、限界（行動は設計、単純な環境、生理学的検証なし）、今後（放出そのものの内発的制御 = 創発、および資源としての場の縮小・多重格納 → `draft_tcds.md` へのポインタ）。

## 9. 他論文との棲み分け

- **本稿（Frontiers）** — 「場が方策をアドレスする、しかもノイズは資源である」という**神経科学向けの機構仮説**。スコープは推論時の選択に限定し、学習則も継続学習も持ち込まない。
- **`draft_nce.md`** — forward ノイズ統計から credit を作る**学習則**の論文。
- **`draft_tcds.md`** — 同じ物理量 $\rho=(\sigma,h)$ を**資源として制御する**（圧縮・複数関数の逐次格納・忘却回避）論文。本稿の「場が選ぶ」の先で「場をどう縮め、どう配るか」を扱う。

3 者は同じ物理量の異なる断面であり、相互引用で一つのプログラムとして読ませる。本稿では $\rho$ の資源制御には踏み込まず、Discussion で一段落だけ触れて `draft_tcds.md` を指す。
