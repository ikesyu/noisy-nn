# 重みパラメータ θ とノイズ場 P の双対性（構想メモ）

NCE 投稿論文（`docs/nce_draft.md`, "Reconstructing Backpropagation from Forward
Fluctuations in Noise-modulated Neural Networks"）の**続編**として、ノイズ場を
主題にする構想の整理。現状の実装（`examples/dual_*.py` 2 本）が NCE 本体の
主張と噛み合っていない点と、その解消方針を記録する。

関連: `docs/forward_noise_covariance_learning.md`（FNCL の技術ノート）、
`docs/recipe_sr.md`（確率共鳴カーブ、Frontiers 向け）。

## 現状（2026-07-17）

§1〜§8 が構想、**§9〜§12 が PoC の実測結果**（`tmp/duality_sigma_grad.py`、
`tmp/duality_gauge_check.py`）。実験で確立したことを先にまとめる。

**まず、設計上の中心的な争点が判明した（§12）。** $h_k = c\,\sigma_k$ の連動は、
恒等式を厳密にする代わりに **σ を重みノルムと冗長にする**。二択になっている:

| | 固定 $h$（NCE 本体の設定） | 連動 $h = c\sigma$（§10.1） |
|---|---|---|
| 恒等式 | est/num = 0.92（8% 過小） | **est/num = 0.996** |
| σ は独立な自由度か | **本物の自由度** | **ゲージ冗長**（§12.1・§12.3） |
| 推定量 | モデルにない対称性を強制（§12.4） | モデルと整合 |
| overlap / mean_P | 健全 | **ゲージ依存で無意味**（§12.6） |

**この二択の解決が続編の設計上の核心である。** 擁護は §12.7（最も強いのは
「ハードウェアでは σ が安い座標だから、冗長でも意味がある」）。

以下、確立したこと:

- **恒等式 $\partial\bar\phi/\partial\sigma = -(d/\sigma)\bar\phi'(d)$ は成立する。**
  交差レイヤ単体で Pearson r = 0.999（ガウス）。さらに $h_k = c\,\sigma_k$ と
  しきい値を場に連動させると系統誤差が消える（est/num: 0.92 → **0.996**）。
  交差が $d/\sigma$ だけの関数になり恒等式が厳密化するためで、`nnn` の変更は不要
  だった（§10.1）。
- **σ 勾配は重み勾配の動径成分であり、独立な第 2 の読み出しではない**（§12.4、
  機械精度で実測）。$\partial L/\partial\log\sigma_k = -(w_k^\mathsf{T}
  \partial L/\partial w_k + b_k\,\partial L/\partial b_k)$。**「1 つの credit に
  2 人の消費者」は「1 つの勾配を 2 つの座標で見ている」が正確。** §1 の「役割が
  直交する 2 つの系」は成り立たず、正しい 3 分解は §12.3。
- **SPSA は σ 勾配とほぼ同じ方向を見ているが、「違いは分散だけ」ではない**（§11.2）。
  cos(SPSA$_K$, σ_grad) の漸近値を固定せずに測ると **$c_\infty = 0.94$ で 1 に
  届かない**（曲線当てはめ 0.9405 と、当てはめを使わない標本モーメント推定 0.9349 が
  独立に一致）。**σ 勾配は真の場の勾配に対して cos ≈ 0.95 の忠実度**で、本体 §5.3 の
  重み勾配（0.998）より低い。分散項ではない（§11.3、統制実験）。**最有力の説明は
  §12.5**: σ 勾配は重み勾配の**動径成分だけ**を読み、その成分は勾配ノルムの
  2〜4 割しかないので、全体の cosine が 0.998 でも射影後は落ちる。
- **予算の差は残る。ただし正しい言い方は「両者を同じ真の勾配に対して測る」こと。**
  SPSA が σ 勾配と同じ忠実度に到達するには **1700〜7000 回の追加 forward**
  （3 seed、SPSA を最強化した CRN 込み）が要り、**σ 勾配は 0 回**。§10.5 の
  「主図は cos(K) 曲線」という判断は変わらないが、$c_\infty$ の水平線と
  2 パラメータ当てはめを載せる必要がある（§11.6）。**最も堅い形は「任意のゼロ次法は
  $\Omega(d)$ 回の追加評価を要し、σ 勾配は 0 回」**（§11.6。座標差分なら $2d$ で
  済むので、SPSA との比較だけを前面に出すと反論される）。
- **$v$ の下限は場の次元に比例する**。理論上 $v = (d+1) + d\,\sigma_\eta^2/|g|^2$ で
  （ガウス摂動。Rademacher なら $d-1$）、損失評価ノイズを $T\to\infty$ で消すと
  $v \to 74.6$ と理論値 65 にほぼ一致した（§11.4）。**SPSA の予算の下限は場の次元と
  ともに増え、σ 勾配は $d$ によらず 0 のまま**。ただし運用点での $v$ は下限の
  3.8〜7.9 倍で、この倍率は $|g|^2$ 依存なので「$v \propto d$ を一定係数で」とは
  言えない。
- **場の分化（H3）は $h$ 固定でのみ主張できる**。σ 勾配は overlap を 0.83 → 0.30 に
  落とし、層 1 の場は P_A / P_B にほぼ排他的に分かれる（§9.4）。**連動下の §10.3 の
  overlap 列は使えない**（§12.6）。加えて overlap 正則化が分化を直接報酬にしている
  ので、低い overlap 自体は自発的なモジュール発見の証拠にならない（§12.9）。
- **credit の質は P 側でも効く（H2）**。`cov_deriv` credit は overlap を逆行させる
  のに対し、`cov_jac` credit は単調（§9.4）。
- **H1 は、$h$ 連動と移動量固定の 2 つの統制を入れると成立する**（§10.3）。
  σ 勾配は SPSA を **21%（ガウス）/ 31%（一様）** 上回る。§9 で立たなかったのは
  提案の欠陥ではなく、固定 $h$ による推定量の劣化と、「SPSA が場を動かさない
  こと」を優位と誤認する統制の甘さが原因だった（§10.4 のアブレーション）。
  損失はゲージ不変なので、この結論は §12 の影響を受けない。

## 1. 双対性とは何か

NNN の出力を

$$y = f(x; \theta, \mathcal{P})$$

と書いたとき、学習可能な自由度が 2 つの異質な系に分かれている、というのが
出発点である。

- **θ**（重み行列群）: 各ユニットの入出力の**中身**を作る。「どのニューロンが
  何を計算するか」を決める、密で連続な線形写像。
- **P**（ノイズ場、ユニットごとのノイズ強度）: どのニューロンが確率的に
  活性化する、すなわち**機能するか**を決める。交差活性
  $\bar\phi(d;\sigma) = 2F(d;\sigma)(1-F(d;\sigma))$ を通して、$\sigma \to 0$ の
  ユニットは応答が恒等的に 0 になり、適切な $\sigma$ を持つユニットだけが
  確率共鳴で信号を伝える（`nce_draft.md` §3.2）。

θ は「計算の実体」、P は「その実体のどこを起動するかというルーティング／
ゲート」を担う。両者は同じ関数 $f$ を規定するが、役割が異なる。
これが双対性の直観である。

> **重要（§12 で判明）。この「役割が直交する 2 つの系」という言い方は、
> 正確ではない。** $h$ を場に連動させた設定（§10.1）では
> $(a_k, \sigma_k) \to (\alpha_k a_k, \alpha_k \sigma_k)$ が厳密なゲージ対称性に
> なり、σ は重みノルムと**冗長**になる。正しい 3 分解は §12.3 にある:
> **重みの方向**（何に反応するか＝θ のみ）、**重みと σ の共通スケール**
> （冗長なゲージ自由度）、**コンテキスト間の相対的な σ**（どこをどれだけ動員するか
> ＝ゲージ不変な実体）。P の作用は θ の作用の**部分空間**であって、直交はしない。
> 以下の §1 の記述は直観としては読めるが、数学的な定式化は §12 に従うこと。

### 双対性を支えている 3 つの構造的性質

1. **分離可能性**。θ を固定したまま P を切り替えるだけで、同一
   ネットワークが別のタスクを解ける。`P_A → sin(x)`、`P_B → sin(2x)` を 1 つの
   θ の上に多重化できる。P はシンボリックな「鍵」ではなく、確率共鳴の最適強度を
   持つ機能的資源として振る舞う（`recipe_sr.md` の逆 U 字と整合）。
   （§12.3: 連動下でこの多重化を担うのは 2 つの場の**比** $\sigma_B/\sigma_A$ で
   あり、共通成分はゲージである。多重化そのものは残るが、担い手が変わる。）
2. **対称性（どちらも独立に学習信号を運べる）**。θ を凍結して P だけ動かしても
   タスク損失が下がるなら、P それ自体が学習信号を担えることの証拠になる。
3. **相補性（joint 最適化での役割分担）**。θ が局所的な入出力マップを作り、
   P がどのマップを起動するかを選ぶ。鍵となるシグネチャは
   **overlap(P_A, P_B) が下がりながら total loss も下がる**こと。

## 2. 現状の実装

| | `examples/dual_node_perturbation_noise_field.py` | `examples/dual_finite_difference_node_perturbation.py` |
|---|---|---|
| θ の更新 | Adam + backprop | 有限差分ノード摂動（出力層のみ解析勾配） |
| P の更新 | SPSA | SPSA（層ごとリスト） |
| モデル | `SimpleNNNStatistic` | 専用 `NNNfd`（`CrossingAnalytic` を利用） |
| rho 初期化 | ランダム | 前半／後半バイアス |
| 追加可視化 | なし | 条件×層ごとの最終ノイズ場 P_A/P_B 棒グラフ |

両者とも 3 条件（theta-only / field-only / dual）を同一の初期 θ・初期 rho から
比較する。P は $P = \mathrm{softplus}(\rho)$ で正値性を保証し、損失は
タスク MSE に sparsity（$\lambda_\text{sparse}\,\mathrm{mean}(P)$）と
overlap（$\lambda_\text{overlap}\,\cos(P_A, P_B)$）の正則化を加えたもの。

### SPSA（Simultaneous Perturbation Stochastic Approximation, Spall 1992）

勾配を計算せず、損失の評価だけから勾配を推定する手法（`nce_draft.md` §2.3 の
関連研究に既出）。パラメータが $d$ 次元のとき素朴な差分は $2d$ 回の損失評価を
要するが、SPSA は全成分を同時にランダム方向 $\delta$ へ振って **2 回**で済ませる：

$$s = \frac{L(\rho + \varepsilon\delta) - L(\rho - \varepsilon\delta)}{2\varepsilon},
\qquad \hat{g} = s\,\delta$$

$s \approx \nabla L \cdot \delta$ より
$\mathbb{E}[\hat g] = \mathbb{E}[\delta\delta^\mathsf{T}]\nabla L = \nabla L$ で、
1 次までの不偏推定量となる（対称差分なので 2 次項が消え、誤差は $O(\varepsilon^2)$）。
分散は大きいが期待値が正しいので、小さい学習率で多数回まわせば降下する。
「$d$ 次元の勾配を 1 個のスカラーとランダム方向の外積で代用する」のが要点。

`dual_finite_difference_node_perturbation.py` の θ-step（ノード摂動）も、
摂動先がパラメータでなくプリ活性 $d$ である点が違うだけで骨格は同一である。

## 3. 問題: 現状の実装は NCE 本体の主張と噛み合っていない

NCE 論文の中心的な発見は、**「大域スカラー損失 × 摂動」型の credit
（`cov_only` / `cov_deriv`、ノード摂動の系譜）は消えない系統誤差を持ち、
backprop に届かない**ことだった（§4.2、§5.2）。

| 手法 | 最終 MSE |
|---|---|
| backprop | 0.00057 |
| `cov_only` | 0.09583 |
| `cov_deriv` | 0.03464 |
| `cov_jac` | 0.00056 |

この線形化バイアスは有限サンプル効果ではないため $T$ を増やしても消えない。

**ところが双対性の 2 スクリプトは、θ 側も P 側もまさにこの「負けた側」の
推定量で作られている。** しかも
`dual_finite_difference_node_perturbation.py:279` の `s_A` は
`F.mse_loss` が全 $N$ を平均済みのため**バッチ全体で 1 個のスカラー**であり、
per-input ですらない完全な pooled 版になっている。NCE のアブレーション
（per-input credit が決定的）からすると、最も弱い変種を使っていることになる。

すなわち、NCE 本体で「これでは駄目だ」と論証した道具立てで、その続編を
書こうとしている状態にある。

## 4. 方向 A（本命）: ノイズ場の勾配を共分散フレームワークの内側から出す

### 恒等式

任意のスケール族 $p(\xi) = \frac{1}{\sigma}q(\xi/\sigma)$ について、

$$\boxed{\;\frac{\partial \bar\phi(d;\sigma)}{\partial \sigma}
= -\frac{d}{\sigma}\,\bar\phi'(d)\;}$$

が成り立つ。導出は $F(d;\sigma) = Q(d/\sigma)$ より

$$\frac{\partial F}{\partial \sigma} = q(d/\sigma)\cdot\left(-\frac{d}{\sigma^2}\right)
= -\frac{d}{\sigma}\,p(d)$$

これを $\partial\bar\phi/\partial\sigma = 2(1-2F)\,\partial F/\partial\sigma$ に
代入すると、$\bar\phi'(d) = 2(1-2F)p(d)$ より直ちに得られる。

**数値検証済み**（`scratchpad/check_sigma_identity.py`、2026-07-15）:
ガウス $\sigma=0.5$ で残差 3.0e-11、一様 $r=1.0$ で 1.3e-10（量のスケールは
0.6〜0.8）。ガウス・一様の双方で機械精度で一致する。

### なぜ効くか

右辺の $\bar\phi'(d)$ は、**論文が §3.1 で既に分布フリーに推定している当の量
$\phi_T'$（KDE slope = 交差計数）**である。$-d/\sigma$ は各ユニットが自分で
持っている量。したがってノイズ場の勾配は、credit $\hat g$ を `cov_jac` から
そのまま流用して

$$\frac{\partial L}{\partial \sigma^{(l)}_k} \approx
\Bigl\langle\,
\underbrace{\hat g^{(l)}_{n,k}}_{\text{第 3 因子（credit）}}
\cdot
\underbrace{\Bigl(-\tfrac{d^{(l)}_{n,k}}{\sigma^{(l)}_k}\Bigr)\phi_T'\bigl(d^{(l)}_{n,k}\bigr)}_{\partial z_k/\partial \sigma_k\ \text{（局所感度）}}
\,\Bigr\rangle_{m,n}$$

と書ける。これは `nce_draft.md` 末尾の三要素則の式

$$\frac{\partial L}{\partial W^{(l)}_{ij}} \approx
\bigl\langle \hat g^{(l)}_{n,i}\cdot\phi_T'(d^{(l)}_{n,i})\cdot z^{(l-1)}_{n,j}\bigr\rangle_{m,n}$$

と**まったく同じ形**で、前シナプス因子 $z^{(l-1)}_j$ が $-d_k/\sigma_k$ に
置き換わっただけである。新しい統計量も、専用の摂動フェーズも、SPSA も要らない。
$\phi_T'$ が分布フリーなので**この $\sigma$ 勾配も分布フリー**であり、一様ノイズでも
同じ回路がそのまま動く（§6.2 の主張がそのまま継承される）。

### 主張の書き換え

> **θ と P は別々の学習機構で動く 2 つの系ではない。同一の credit $\hat g$ を、
> 外積の相手として $z_\text{prev}$ に当てれば θ の勾配、$-d/\sigma$ に当てれば
> P の勾配が出る。1 つの前向き共分散 credit に、2 人の消費者がいる。**

SPSA／ノード摂動版は**手法ではなくベースライン・アブレーション**に格下げし、
「摂動ベースでもできるが、共分散 credit を使えば桁で良い」と示すのが正しい配置。
NCE 本体の `cov_deriv` vs `cov_jac` の対比が P 側でも再現するはずで、これ自体が
結果になる。

## 5. 方向 B: NCE 本体には入れず、続編として書く（既に予告済み）

ドラフトは双対性を入れる場所を自分で空けている：

- **§3.2（240 行目）**: 「$s_k = 0$ のユニットは共分散 credit も厳密に 0 となって
  更新から自動的に外れる。これは **credit assignment と recruitment が同一の場を
  共有する**ことを意味し…」
- **§6.2 末（583〜585 行目）**: 「ノイズは分布だけでなく**空間的な割り当ても
  設計変数**である。…これらの設計とその評価は future work とする。」
- **§7（601〜603 行目）**: 「ノイズ場は、**学習資源の動員と消費電力の双方を
  空間的に制御する変数**であり…既存研究にない新しい視点を与える。」

NCE 本体は「backprop の再構成」で主張が既に一杯であり、双対性を足すと焦点が
ぼやける。**予告を回収する続編**という位置づけが素直で、§3.2 の
「credit と recruitment が同一の場を共有する」という一文が続編の主張そのものの
種になっている。方向 A を採れば、続編は「NCE で作った credit をノイズ場に
向けるだけで、場も学習できる」という最小の増分で最大の主張になる。

## 6. 方向 C: 「1 つの場が推論・学習・電力を同時にゲートする」を工学側の見出しに

方向 A から自動的に出る系として、$\sigma_k$ が学習可能になると $s_k \to 0$ は
**学習の結果として**到達できる。そのとき §3.2 の性質により、そのユニットは
推論からも学習からも厳密に外れる。さらに §6.2 の**一様ノイズのコンパクト
サポート**（$|d-c| \ge r$ で発火も学習統計も厳密に 0）と組み合わせると：

> ノイズ場という単一の変数が、(i) どのサブネットワークが計算するか、
> (ii) どこに credit が流れるか、(iii) どこが電力を食うか、を同時に決める。
> しかもそれ自身が同じ credit で学習される。

抽象的な「双対性」より、この具体的な帰結を前面に出すほうが査読で強い。
§7 が期待する FPGA + 小型ロボットの筋にそのまま接続する。

## 7. 方向 D: Frontiers 投稿との棲み分け

Frontiers「Computational Models of Neuromodulation」特集（締切 2026-09-11）も
同じノイズ場を主役にする計画であり、放置すると 2 本が競合する。切り分け案：

- **Frontiers** = 神経修飾の枠組み。主張は「ノイズ場が共有重みに多重化された
  方策を*アドレスする*、最適強度を持つ機能的資源（SR）」。主図は
  `examples/sr_separation_curve.py --sweep train --model sample` の逆 U 字。
  **場は与えるもの**で、学習則の話はしない。
- **NCE 続編** = 学習則の枠組み。主張は「その場自身が、backprop を再構成したのと
  同じ前向き共分散 credit で学習される」。多重化（P_A/P_B）は場が学習可能で
  あることの**デモ**であって、主張は credit の側にある。

この切り分けなら、Frontiers が「場は資源である」を、続編が「場は学習される」を
担い、相互に引用できて競合しない。

## 8. 方向 E: 現行スクリプトへの具体的な修正

1. **θ-step をノード摂動から `cov_jac` credit へ**（方向 A）。
   `data_nce/fncl/train.py` の `train_cov` / `covariance_credit` / `cov_weight` が
   そのまま使える。
2. **P-step を SPSA から $\sigma$ 勾配へ**（方向 A）。SPSA は比較対象として残す。
3. **pooled スカラー credit をやめる**。現状 `s_A` はバッチ全体で 1 スカラー。
   NCE のアブレーションは per-input が決定的だと示している。
4. **`SimpleNNNStatistic` → `SimpleNNNSample`**。Analytic/Statistic は平均場
   （理論）、Sample が機構。NCE 本体も §3.2 で「本研究の対象は Sample-level」と
   明示しており、続編もそこに揃えるべき。
5. **`softplus(rho)` パラメータ化の再検討**。$\sigma$ 勾配が直接出るなら rho を
   経由する必然性は薄い。一様ノイズなら半径 $r$ を直接クリップするほうが §6.2 の
   ハードウェア議論と整合する。

## 9. PoC の結果（`tmp/duality_sigma_grad.py`、2026-07-16）

**$h$ が結果を反転させる。** ガウスで $h$ を 0.2 → 0.1 にするだけで `sigma_grad` の

兼ねるため、場が細くなるほど推定が破綻する。続編では $h$ と σ の関係を明示的に
設計する必要がある（例: $h$ を場に比例させる）。

**σ 勾配は場を一律に縮める。** mean_P はどの設定でも 0.50 → 0.23〜0.28 へ落ちる。
λ_sparse=0 にしても落ちる（0.232）ので、**sparsity 正則化は原因ではなく、
タスク損失の σ 勾配自体が縮める方向を向いている**。ただし σ→小 はユニットの死では
なく**受容野が鋭くなる**ことを意味し（1 次元回帰では有利な基底）、損失は悪化しない。
一様 $h$=0.1 の勝ちパターンでも P_B の層 2 は床に達しているが損失は最良だった。
この「場が勝手に鋭くなる」挙動が §6 の電力ゲーティングの主張とどう噛み合うかは、
続編で詰めるべき論点である。

**公平性の留保。** SPSA は場の移動量が小さく（mean_P 0.50→0.42、overlap 0.83→0.65）、
θ が補償している可能性がある。同一の場の移動量で揃えた比較が要る。

## 10. 第 2 次 PoC: $h$–σ 連動と、移動量を揃えた比較（2026-07-16）

§9.5 で挙げた 2 つの課題を実行した。

### 10.1 $h$ を σ に連動させると恒等式が厳密になる

**設計**: 交差しきい値をユニットごとに $h_k = c\,\sigma_k$ と場に比例させる
（`--couple-h C`）。`nnn` の `CrossingSample` はテンソルの $h$ をそのまま
ブロードキャストするので、ライブラリ側の変更は不要だった。

**理由（これが要点）**: $h \propto \sigma$ にすると交差の応答は
$\bar z(d;\sigma) = g(d/\sigma)$ という **$d/\sigma$ だけの関数**（スケール族）に
なる。すると

$$\frac{\partial \bar z}{\partial \sigma} = g'\!\left(\frac{d}{\sigma}\right)\cdot\left(-\frac{d}{\sigma^2}\right)
= -\frac{d}{\sigma}\cdot\frac{\partial \bar z}{\partial d}$$

が **$c$ の値によらず厳密に**成立する。固定 $h$ では $h$ が第 2 のスケールを
持ち込むため $\bar z$ が $d/\sigma$ だけの関数でなくなり、そこから $O(h^2)$ 誤差が
生じていた。

**実測**（レイヤ単体、`--check`）。固定 $h$ では **est/num ≈ 0.92 の系統的な
8% 過小評価が $h$ を変えても消えない**のに対し、連動させると 1.0 になる:

| 設定 | Pearson r | est/num |
|---|---|---|
| ガウス 固定 $h$=0.2 | 0.9882 | 0.924 |
| ガウス 固定 $h$=0.1 | 0.9987 | 0.922 |
| ガウス **連動 $c$=0.2** | 0.9987 | **0.996** |
| ガウス **連動 $c$=0.1** | 0.9991 | **0.992** |
| 一様 固定 $h$=0.1 | 0.8950 | 0.881 |
| 一様 **連動 $c$=0.2** | 0.8343 | **1.017** |

**残る限界（一様）**: 一様ノイズは $\bar\phi' = -(d-c)/r^2$ がサポート端で
不連続なので $\bar\phi''$ にデルタが立ち、$O(h^2)$ 誤差が折れ点に集中する。
連動はスケール（est/num）を直すが折れ点は直せず、r は 0.83〜0.90 に留まる。
ガウスは $\bar\phi$ が滑らかなので連動で r=0.999・比 0.996 と、ほぼ完全になる。

### 10.2 SPSA は分散を払って σ 勾配の方向へ近づく

**設計**（`--compare-dir`）: 同一の状態で σ 勾配と、$K$ 個の独立な SPSA 推定の
平均を計算し、その cosine を $K$ の関数として測る。SPSA は不偏なので $K\to\infty$ で
真の場の勾配に収束する。したがって cosine を $K$ の関数として見れば、**SPSA が
どれだけ分散を払っているか**と、**σ 勾配が真の勾配とどれだけ方向が合っているか**を
分離して測れる。実測は

$$\cos(K) = \frac{c_\infty}{\sqrt{1 + v/K}}$$

の形によく従い、$K$ とともに単調に上昇する。**$v$ が前者（SPSA の分散）、
$c_\infty$ が後者（σ 勾配の忠実度）であり、両者は別のことを言っている。**

**定量的な結果は §11 にある。** この節の初回測定は $K$=256 までで、しかも
$c_\infty = 1$ を**仮定して**当てはめていた（$v$=388、cos=0.9 に 3310 forwards）。
$c_\infty$ は測るべき量であって仮定してよい量ではなく、自由にして測ると 0.94 で
1 に届かない。**「違いは分散だけ」は成り立たない。** §11.2 が $K$=1024 までの直接
測定と、当てはめに依存しない標本モーメント推定でこれを確定させたので、
**数値は §11.2 の表を使うこと**（参照 σ 勾配の平均パス数なども増やしてあり、
この節の初回測定とは値が異なる）。

**σ 勾配は、推論のために既に引いている $T$=32 サンプルから方向を読む。
追加 forward は 0 回である。** この核心（予算の非対称性）は §11 でも変わらず、
§9 の H1（損失で勝つ）より強い主張であるという §10 の判断もそのまま残る。

**§9 の H1 の再解釈**: 学習ループの SPSA は $K$=1、すなわち **cos ≈ 0.05 の
ほぼランダムな方向**に動いている（§11.2）。それでも損失で勝っていたのは、方向が良い
からではなく、ランダムウォークで場が実質ほとんど動かず（mean_P 0.50→0.42）、
θ が補償していたためと考えられる。「SPSA が勝つ」は「SPSA が場を動かさない」
とほぼ同義だった。

### 10.3 移動量を揃えると H1 が成立する

**設計**（`--matched-step S`）: Adam をやめ、毎エポックの場の更新を
ノルム $S$ に正規化してから適用する。これにより **SPSA も σ 勾配も場を
1 エポックあたり厳密に同じ距離だけ動かす**ので、差は推定の**方向**だけに帰着する。
§10.2 が示したとおり SPSA の見かけの優位は「動かないこと」だったので、
この統制が H1 の本来の検証にあたる。

**ガウス（$c$=0.2 連動、$S$=0.01、1000 epoch, H=32, T=32, seed 0）**:

| 条件 | total MSE | overlap | mean_P |
|---|---|---|---|
| `theta_only` | 0.1235 | 0.834 | 0.502 |
| `spsa` | 0.0886 | 0.759 | 0.472 |
| `sigma_cov_deriv` | 0.1149 | 0.442 | 0.390 |
| **`sigma_grad`** | **0.0703** | 0.467 | 0.396 |

**H1 成立**: σ 勾配が SPSA を **21% 上回る**（0.0703 vs 0.0886）。しかも
mean_P は 0.40 vs 0.47 と同水準で、場の移動量が揃っている（統制が効いている）。
**H2 も成立**: `cov_jac` credit は `cov_deriv` credit を **39%** 上回る
（0.0703 vs 0.1149）。

**一様（$c$=0.2 連動、$S$=0.02）でも同じ結論**:

| 条件 | total MSE | overlap | mean_P |
|---|---|---|---|
| `theta_only` | 0.0841 | 0.834 | 1.004 |
| `spsa` | 0.0666 | 0.819 | 0.984 |
| `sigma_cov_deriv` | 0.0658 | 0.534 | 0.827 |
| **`sigma_grad`** | **0.0459** | 0.558 | 0.837 |

σ 勾配が SPSA を **31%**、`cov_deriv` credit を **30%** 上回る。**H1・H2 が
ガウス・一様の両方で同時に成立した**（H4）。

**この 2 表の overlap 列と mean_P 列は使えない。** どちらも $c$=0.2 連動での実験
であり、連動下では overlap と mean_P はゲージ依存量、すなわち**同一の関数を計算する
パラメータ設定どうしで値が変わる**（§12.6 で実測）。したがってここでの H3 は
主張できない。**H3 の証拠は $h$ 固定で測った §9.3・§9.4 の表**（ゲージ対称性が
存在しないので健全）であり、連動下で分化を主張したいなら §12.6 のゲージ不変な
指標で測り直す必要がある。損失（total MSE）は関数の量なのでゲージ不変であり、
H1・H2 の結論は影響を受けない。

### 10.4 2 つの修正はそれぞれ別の仕事をしている（アブレーション）

ガウスで設定を 1 つずつ足すと、効果が綺麗に分解できる（`spsa` vs `sigma_grad`）:

| 設定 | spsa | sigma_grad | 判定 |
|---|---|---|---|
| 固定 $h$=0.2, Adam（§9 既定） | **0.0755** | 0.1313 | σ 勾配の負け |
| 固定 $h$=0.1, Adam | **0.0615** | 0.0649 | 実質互角 |
| 連動 $c$=0.2, Adam | **0.0690** | 0.0723 | 実質互角 |
| **連動 $c$=0.2, 移動量固定 $S$=0.01** | 0.0886 | **0.0703** | **σ 勾配の勝ち** |

**$h$ 連動は推定量を直す**: `sigma_grad` を 0.1313 → 0.0723 と **45% 改善**する。
§10.1 のとおり恒等式の系統誤差（8% 過小）が消えるためで、SPSA はほぼ影響を
受けない（0.0755 → 0.0690）。

**移動量固定は交絡を除く**: `sigma_grad` はほとんど変わらない（0.0723 → 0.0703）
のに対し、**SPSA だけが悪化する**（0.0690 → 0.0886）。これは §10.2 の直接の帰結で
ある。**SPSA は $K$=1 で cos ≈ 0.01 のほぼランダムな方向を向いているので、実際に
場を動かすことを強いると損をする。** σ 勾配は方向が正しいので、同じ距離を歩かされ
ても損をしない。

つまり **§9 で H1 が立たなかったのは提案の欠陥ではなく、(i) 固定 $h$ による推定量の
劣化と (ii) 「SPSA が場を動かさないこと」を優位と誤認する統制の甘さ、という 2 つの
交絡が原因だった。** 両方を除くと H1 はガウス・一様の双方で成立する。

なお移動量を揃えると、σ 勾配の overlap は 0.30 → 0.47（ガウス）/ 0.56（一様）と
§9 より高くなる。これは 1 エポックあたりの変位を制限した当然の帰結であり、
それでも SPSA（0.76 / 0.82）よりはるかに低い。**分離の速さと損失の低さを同時に
達成している。**

### 10.5 結論と、続編に向けて残る課題

**方向 A は、推定量としては成立する。** 恒等式は厳密（$h$ を場に連動させれば
est/num = 0.996）、SPSA は分散を払って σ 勾配の方向へ近づき、公平な比較では
損失・分離の双方で SPSA と `cov_deriv` credit を上回る。しかも追加 forward は 0 回。

**主張として最も強いのは §10.2 の予算の非対称性**であり、損失の勝敗より頑健で
設定に依存しない。続編の主図はこの cos(K) 曲線にすべきである（図の作りは §11.6）。

**ただし §10.1 の $h$ 連動には、当時気づいていなかった代償があった。**
$\bar z$ が $d/\sigma$ だけの関数になるということは、$(a_k, \sigma_k) \to
(\alpha_k a_k, \alpha_k \sigma_k)$ というゲージ対称性が生じるということでもある。
σ は重みノルムと冗長になり、「θ と P は役割が直交する 2 つの系」という §1 の
枠組みはこの設定では成り立たない。**§12 がこの帰結を扱う。§10・§11 の
overlap と mean_P の数値は、連動下ではゲージ依存量である**（§12.6）。

残る課題（§11.7・§12.7 に更新版がある）:

1. **seed 1 本のみ**。上の数値は seed 0 の単一試行で、seed 間ばらつきは未測定。
   本体 §5.1 と同じ seeds 0–2 での再実行が要る。
   （§11.2 で cos(K) 側だけ 3 seed を測った。$c_\infty$ が 0.82〜0.97、$v$ が
   177〜494 と**ばらつきは大きい**。学習比較の 3 seed 化は未実施のまま。）
2. **一様は折れ点で r が 0.83〜0.90 に留まる**（§10.1）。ガウスは 0.999。
   ハードウェア指向（本体 §6.2）は一様を選ぶので、この差の含意を詰める必要がある。
3. **$c$ と $S$ は未探索**。$c$=0.2、$S$=0.01/0.02 を 1 点ずつ試したのみ。
4. **$h$ 連動の副作用**。$h$ は交差しきい値そのものなので、$h \propto \sigma$ は
   ハードウェアでは「しきい値を場に追従させる回路」を意味する。本体 §6 の
   プリミティブ表（比較器 2 個＋XOR）に何を足すのかを見積もる必要がある。
5. **σ が縮む挙動と §6 の電力ゲーティングの関係**（§9.5 から継続）。

## 11. 第 3 次 PoC: cos(K) の漸近値を測る（2026-07-16）

§10.2 は $\cos(K) = 1/\sqrt{1+v/K}$ と、**$K\to\infty$ で $\cos\to1$ を仮定した**形で
当てはめていた。しかし §9.7 が自分で書いたとおり σ 勾配と SPSA は推定量が違う
（SPSA は $E[L]$ 全体、σ 勾配は信号項のみ）ので、漸近値は原理的に 1 未満のはずである。
また実測は $K$=256 までで、cos=0.9 の 3310 forwards は 6.5 倍の外挿だった。
漸近値を自由パラメータにし、$K$=1024 まで直接測って両方を確かめた。

### 11.1 cos(K) の 2 つのパラメータは別々のことを言っている

SPSA の 1 ドローは $\hat g = (g^\mathsf{T}\delta)\delta + \eta\,\delta$ と書ける
（$\eta$ は損失評価ノイズが差分商に入る項）。ガウス $\delta$ に対し
$E[(g^\mathsf{T}\delta)^2\delta\delta^\mathsf{T}] = |g|^2 I + 2gg^\mathsf{T}$ なので

$$\mathrm{Cov}(\hat g) = |g|^2 I + gg^\mathsf{T} + \sigma_\eta^2 I,
\qquad \mathrm{tr}\,\Sigma = (d+1)|g|^2 + d\,\sigma_\eta^2$$

$\mathrm{SPSA}_K = g + \tilde e$（$\mathrm{Cov}(\tilde e) = \Sigma/K$）だから、
デルタ法で

$$\cos(K) \approx \frac{c_\infty}{\sqrt{1 + v/K}},\qquad
c_\infty = \cos(g_\text{true},\, \hat g_\sigma),\qquad
v = \frac{\mathrm{tr}\,\Sigma}{|g|^2} = (d+1) + \frac{d\,\sigma_\eta^2}{|g|^2}$$

**傾き $v$ は SPSA が払っている分散、漸近値 $c_\infty$ は σ 勾配が真の勾配と
どれだけ方向が合うかという忠実度**であり、別々のことを言っている。$c_\infty$ を
1 に固定することは、後者を測らずに仮定することにあたる。

この形の効用として、**$v$ と $c_\infty$ は曲線当てはめを使わず標本モーメントからも
推定できる**（$M$ ドローに対し $E|\bar g|^2 = |g|^2 + \mathrm{tr}\Sigma/M$ を
引き算して $|g|^2$ を得る）。当てはめと独立な検証になるので、両方を出す。

### 11.3 ギャップの出所は分散項「だけ」ではない（統制実験）

$E[L] = (E[y]-t)^2 + \mathrm{Var}(y)$ で、アンサンブル平均を読むので
$\mathrm{Var}(y) = \mathrm{Var}_1/T$。σ 勾配は $\bar z$（平均応答）の微分なので
信号項しか推定しない。**分散項が原因なら、$T$ を増やせば $c_\infty \to 1$ となるはず。**

θ・場・参照 σ 勾配をすべて固定し、**SPSA が損失を平均する $T$ だけ**を振った
（seed 0、`--eval-t-sweep`）。各 $T$ で θ を訓練し直すと交絡するので、それを避ける
のがこの統制の要点である。

| 評価 $T$ | $v$ | $c_\infty$ | 参照にも同じ $T$ を与えた場合 |
|---|---|---|---|
| 8 | 1031.4 | 0.794 | 0.826 |
| 32（運用点） | 297.6 | 0.937 | 0.941 |
| 128 | 97.8 | **0.958** | 0.961 |
| 512 | 72.3 | **0.941** | 0.947 |

- **$c_\infty$ は 0.79 → 0.94 → 0.96 と上がるが、0.95 前後で頭打ちになり 1 に
  行かない。** 分散項は $T$=8→32 の上昇分を説明する。しかし残りは説明しない。
- **参照 σ 勾配にも同じ $T$ を与えても変わらない**（0.937 → 0.941、0.958 → 0.961）。
  credit は $T$ サンプルの共分散の比なので有限 $T$ バイアスを持つはずだが、それが
  主因ではないということ。**残差は構造的**である。
- したがって **σ 勾配の忠実度は、それ自身が推定している信号項に対してすら
  cos ≈ 0.95** にとどまる。**本体 §5.3 の重み勾配 0.998 より明確に低い。**
  これは続編で正直に報告すべき制約であり、隠すと査読で必ず出る。
  **出所の最有力候補は §12.5**: σ 勾配は重み勾配の**動径成分だけ**を読んでおり
  （§12.4 の恒等式）、その成分は勾配ノルムの 2〜4 割しかない。全体の cosine が
  0.998 でも、低次元射影の cosine がそれより大きく落ちるのは自然である。

### 11.4 $v$ の正体は場の次元（理論の直接検証）

上の表の $v$ 列が §11.1 の理論をそのまま検証している。$T\to\infty$ で損失評価
ノイズ $\sigma_\eta^2$ が消えるので $v \to d+1$ となるはずで、**実測は $T$=512 で
$v = 72.3$、理論値 $d+1 = 65$ にほぼ一致した**。$v$ の下限は場の次元そのものである。

次元スイープ（seed 0、ドロー数を次元に比例させ $v/M \le 0.11$ を確保。この比が
大きいと $|g|^2 = |\bar g|^2 - \mathrm{tr}\Sigma/M$ の引き算が信号を食って $v$ が
上振れするため、初回の $M$=2048 固定では dim=256 が壊れていた）:

| 場の次元 $d$ | $v$（モーメント） | $d+1$ | $v/(d+1)$ | SPSA が σ 勾配に並ぶ追加 forward |
|---|---|---|---|---|
| 16 | 134.2 | 17 | 7.90 | 393 |
| 32 | 168.2 | 33 | 5.10 | 1774 |
| 64 | 248.9 | 65 | 3.83 | 3816 |
| 128 | 500.6 | 129 | 3.88 | 4223 |
| 256 | 1736.3 | 257 | 6.76 | （当てはめ不能） |

**予測は半分当たった。**

- **当たり**: $v$ は次元とともに増える（$d$: 16→256 で $v$: 134→1736）。しかも
  **下限が場の次元に比例することは理論と $T\to\infty$ の実測の両方で確定した**。
  SPSA は損失評価をどれだけ静かにしても、この下限を下回れない。**σ 勾配の予算は
  $d$ によらず 0 なので、両者の差は場が大きくなるほど必然的に開く。**
  ただし **$d+1$ という具体値はガウス方向の摂動に固有**である。古典的 SPSA が使う
  Rademacher（$\pm1$）摂動では $|\delta|^2 = d$ が確定値なので
  $E|\hat g|^2 = d|g|^2$、すなわち $\mathrm{tr}\Sigma = (d-1)|g|^2$ となり下限は
  $d-1$。差は小さいが、**「厳密に $d+1$」を一般の SPSA の下限として書いてはいけない**
  （`--spsa-crn` と同様、ベースラインは Rademacher にすべきである）。
- **外れ**: $v/(d+1)$ は 7.90, 5.10, 3.83, 3.88, 6.76 と**一定にならない**。
  超過分は $\sigma_\eta^2/|g|^2$、すなわち損失ノイズと勾配の大きさの比であり、
  これは幅ごとにネットワークと当てはまり具合が変わると動く。**「$v \propto d$ を
  一定係数で」とは言えない。**主張は「下限が $d+1$」に絞るべき。
- dim=256 は当てはめが不能（$c_\infty$=1.03 と非物理）。$v\approx1736$ に対し
  $K$ が 1024 までだと $\cos \approx c/1.64$ で漸近域に入らず、$c$ と $v$ が
  識別できないため。**大きい $d$ で曲線当てはめを使うなら $K$ を $v$ 以上まで
  伸ばす必要がある。**モーメント推定にはこの制約がない。

### 11.5 CRN: ベースラインを最強にしても結論は変わらない

$v$ が下限の 3.8 倍ある主因は、SPSA の $\pm$ 2 回の forward が**独立に**ノイズを
引いていることである。共通乱数（CRN）で同じ乱数を replay すれば $\sigma_\eta$ は
縮むはずで、これは SPSA ベースラインの公平性の問題でもある（実装しないと
「SPSA を不当に弱くしている」と突かれる）。`--spsa-crn` で実装した。

| | $v$（モーメント） | $c_\infty$（当てはめ / モーメント） | SPSA が σ 勾配に並ぶ追加 forward |
|---|---|---|---|
| 独立ノイズ | 248.9 | 0.9405 / 0.9349 | 4551 |
| **CRN** | **155.6** | **0.9228 / 0.9271** | **1721** |

- **CRN は実際に SPSA を強くする**（$v$ が 249 → 156、下限 65 の 2.4 倍まで）。
  予算も 4551 → 1721 forwards に減る。**続編のベースラインは CRN 版を使うべき。**
- Appendix B・§9.7 が警告した「二値交差でペア差分が退化する」現象は**起きない**。
  N=128 × T=32 のアンサンブル平均で均されるため。
- **$c_\infty$ はほぼ不変**（0.94 → 0.92）。$c_\infty$ は SPSA の実装によらない
  σ 勾配側の量である、という独立の確認になっている。**別々の不偏推定量が同じ
  漸近方向を指し、どちらも σ 勾配をそこから cos≈0.93 の位置に見ている。**

### 11.6 主張の書き換え

「SPSA と σ 勾配の違いは分散だけ」は成り立たない。正しい主張は 3 つ:

1. **σ 勾配は真の場の勾配を cos ≈ 0.95 で再現する推定量である**（信号項に対して。
   本体 §5.3 の重み勾配 0.998 より低い）。これは制約であり、正直に書く。
2. **「SPSA が cos=0.99 に到達する予算」は存在しない。** $c_\infty$ を超えられない
   ため。ただし**これは σ 勾配に有利な事実ではない**。σ 勾配が真の勾配でないことの
   帰結にすぎず、この形で書くと誤導になる。
3. **正しい予算の言い方は、両者を同じ真の勾配に対して測ること。**
   $\cos(\mathrm{SPSA}_K, g_\text{true}) = 1/\sqrt{1+v/K}$ が $c_\infty$ に達する
   $K$ は $v\,c_\infty^2/(1-c_\infty^2)$。seed 0 で 4551 forwards、CRN で
   1721、3 seed で 1657〜7005。**σ 勾配は 0 回。**これは §10.5 の結論
   （主図は cos(K) 曲線）を変えないが、**図には $c_\infty$ の水平線と 2 パラメータ
   当てはめを載せる**必要がある。

$v$ については「$v$ の下限は場の次元に比例し、SPSA はそれを下回れない。
σ 勾配の追加コストは $d$ によらず 0 である」と書くのが、実測で支えられる
最も強い形である。

**ただし予算の比較先を SPSA に限ると反論される。** ランダム方向の平均は
ゼロ次法として最弱の部類で、座標有限差分なら $2d$ forward（$d$=64 なら 128 回）、
直交方向や Hadamard 方向を使えば独立方向より分散をさらに落とせる。
「SPSA が 4551 回」を普遍的なゼロ次法とのコスト差として前面に出すのは危険である。
**堅い形は「任意のゼロ次法は $d$ 個の未知数を決めるのに最低 $\Omega(d)$ 回の
追加評価を要する。σ 勾配は 0 回である」**という情報量的な言い方であり、
これなら手法の選び方に依存しない。

### 11.7 残る課題（§10.5 から更新）

1. **$c_\infty \approx 0.95$ の残差の内訳。** 分散項でも有限 $T$ でもなく構造的だと
   分かった（§11.3）。**§12.5 が最有力の説明（動径射影の低次元性）を与えた**ので、
   次は seed ごとに $r_k = |a_k^\mathsf{T}g_{a_k}|/(\|a_k\|\|g_{a_k}\|)$ を測り、
   $r_k$ が小さい seed ほど $c_\infty$ が低いかを確認する。これが立てば課題 2 の
   seed ばらつきも同時に説明できる。
   **注意: autograd を基準にする素直な手は使えない。** `CrossingAnalytic.backward`
   は入力についての勾配しか返さず σ は CDF のクロージャに焼き込まれている。加えて
   Analytic は平均場なので層 2 以降は Sample モデルと一致しない（§8 の項目 4）。
   現実的な道は、**大きい $T$ での SPSA を「真の信号項勾配」の代理基準として使い、
   credit の変種を差し替えて $c_\infty$ を比べること**（§11.3 の機構がそのまま使える）。
2. **seed ばらつきが大きい。** $c_\infty$ が 0.82〜0.97、$v$ が 177〜494（3 seed）。
   本体 §5.1 と同じ seeds 0–2 でもこの幅なので、続編の図には誤差帯が要る。
3. 一様ノイズでの §11 の再測定は未実施（§11 はガウスのみ）。
4. **ゼロ次参照の強化。** Rademacher 摂動（下限 $d-1$）、座標差分（$2d$）、直交方向、
   $\varepsilon$ sweep。現在の「真の勾配」は有限 $\varepsilon$ の高予算 SPSA でしか
   なく、厳密な真値ではない（§11.6・§12.9）。
5. §10.5 の課題 2〜5（一様の折れ点、$c$ と $S$ の探索、$h$ 連動の HW コスト、
   σ が縮む挙動と電力ゲーティング）は未着手のまま。**ただし 5 番目は §12.6 により
   問いの立て方が変わった**: 連動下では σ の縮小自体がゲージ量なので、実効ゲイン
   $\gamma_k = \|[w_k,b_k]\|/\sigma_k$ で測り直すこと。

## 12. $h$ 連動が生むゲージ対称性（2026-07-17）

外部レビュー（`docs/report1.pdf`）の指摘を検証した結果、**§10.1 で導入した
$h_k = c\,\sigma_k$ の連動が、恒等式を厳密にするのと引き換えに、σ を重みノルムと
冗長にしていた**ことが判明した。以下はすべて `tmp/duality_gauge_check.py` による
実測で、多くが**機械精度で厳密**である。

### 12.1 ゲージ対称性（実測）

$h_k = c\,\sigma_k$ にすると $\bar z$ は $d/\sigma$ だけの関数になる（§10.1）。
これは各隠れユニット $k$ について

$$\boxed{\;a_k \to \alpha_k a_k,\qquad \sigma_k \to \alpha_k \sigma_k
\qquad\Longrightarrow\qquad f \text{ は不変}\;}$$

を意味する（$a_k = [w_k, b_k]$、$\alpha_k > 0$ は各ユニット独立）。ノイズは
$\xi = \sigma u$、しきい値は $h = c\sigma$ として入るので、**同一の乱数の下では
$d$・$\xi$・$h$ がすべて $\alpha_k$ 倍**され、交差は (信号, しきい値) の同時
スケールで不変だから $\alpha_k$ が消える。したがって不変性は期待値ではなく
**経路ごとに厳密**である。

| 操作（$h$=0.2σ 連動） | max\|dy\| | max\|dz^(1)\| |
|---|---|---|
| 層 1 だけをゲージ変換 | **0.00e+00** | **0.0e+00** |
| 層 2 だけをゲージ変換 | **0.00e+00** | **0.0e+00** |
| 両層 | **0.00e+00** | **0.0e+00** |
| 参考: $h$=0.2 固定 | 1.76e-01 | 1.0e+00 |

**固定 $h$ では対称性は存在しない。** $h$ が第 2 のスケールを持ち込むので
$\bar z$ が $d/\sigma$ だけの関数でなくなり、σ は本物の自由度になる。
**つまりこのゲージ問題は NCE 本体の設定にはなく、§10.1 の提案が持ち込んだもの
である。**

### 12.2 なぜ中間層を貫通するのか（局所ゲージだから）

「層 1 のユニットは層 2 の全ユニットへ入力するのだから、中間層があれば壊れるのでは」
という疑問は自然だが、**壊れない**。上表の $dz^{(1)} = 0$ がその答えである。
ゲージはユニットの**出力**を不変に保つので、**下流に伝播すべきものが何も発生しない**。
「下流で効果が打ち消し合う」のではなく、中間表現がビット単位で動かない。

ReLU 網の正斉次性と対比すると違いが際立つ。あちらは層 $l$ の重みを $\alpha$ 倍すると
**出力が $\alpha$ 倍される**ので、**次の層の重み**を $1/\alpha$ 倍して補償せねばならず、
対称性が隣接層の共謀として層をまたぐ。**NNN の連動下では補償が同じユニットの
$\sigma_k$ の中で完結する**ので、対称性がユニットの外に出ない。これは大域的対称性
ではなく**局所ゲージ対称性**であり、だから深さによらず各ユニット独立に成立する。

### 12.3 冗長性の大きさ: 単一コンテキストでは 100%

ゲージ自由度は隠れユニット 1 個につき 1 個。$[1,32,32,1]$ なら隠れユニット 64 個、
場の次元も 64 で、**数が一致する**。実際 $\alpha_k = 1/\sigma_k$ を取ると:

| | mean | range | max\|dy\| |
|---|---|---|---|
| 元の場 | 0.5034 | [0.317, 0.713] | — |
| ゲージ後 | 1.0000 | [1.000, 1.000] | **0.00e+00** |

**学習された場を完全に平坦な $\sigma \equiv 1$ に潰しても、出力はビット単位で
同一である。単一コンテキストの場は 100% ゲージであり、機能的な情報を一切
持たない。**

2 コンテキストでは $a_k$ が共有なので $\alpha_k$ は 1 個しかなく、$\sigma_{A,k}$ と
$\sigma_{B,k}$ は連動して動く。よって場のパラメータ 128 個のうち **64 個がゲージ、
残り 64 個（比 $\sigma_{B,k}/\sigma_{A,k}$）が実体**である。**多重化のデモは
半分生き残る。** 正しい 3 分解は:

1. **重みの方向 $\hat a_k$**（$d-1$ 次元）: 何に反応するか。**θ のみ**が動かせる。
2. **共通スケール**（1 次元）: 冗長なゲージ自由度。
3. **コンテキスト間の相対 σ**: どのコンテキストでどれだけ動員するか。**ゲージ不変**。

すなわち **P が動かせるのは各ユニットの動径（ゲイン）1 次元だけで、方向は θ に
しか動かせない。P の作用は θ の作用の部分空間であり、直交しない。**

### 12.4 ゲージ恒等式: σ 勾配は重み勾配の動径成分（実測）

不変性から Euler の関係が従う:

$$\frac{\partial L}{\partial \log \sigma_k}
+ \Bigl(w_k^\mathsf{T}\frac{\partial L}{\partial w_k}
+ b_k \frac{\partial L}{\partial b_k}\Bigr) = 0$$

これは代数的に PoC の推定量でも確かめられる。$\delta_k = \hat g_k \phi_T'(d_k)$ と
おくと $w_k^\mathsf{T} g_{w_k} + b_k g_{b_k} = \langle \delta_k (w_k^\mathsf{T}
z_\text{prev} + b_k)\rangle = \langle \delta_k d_k\rangle$ であり、σ 勾配は
$\langle \delta_k \cdot(-d_k/\sigma_k)\rangle$ だから、両者は厳密に符号反転の関係にある。
実測（残差）:

| | 層 1 | 層 2 |
|---|---|---|
| $h$=0.2σ 連動 | 8.15e-09（相対 5.09e-07） | 1.86e-09（相対 5.87e-08） |
| $h$ 固定 | 8.15e-09（相対 5.09e-07） | 1.86e-09（相対 5.87e-08） |

**$\sigma$ 勾配は、同じ credit から出る重み勾配の動径成分の符号反転そのもの**で
あり、独立な第 2 の読み出しではない。「1 つの credit に 2 人の消費者」は、
**「1 つの勾配を 2 つの座標で見ている」**が正確である。

**そして $h$ 固定でも数値が完全に同一であることが重要**である。推定量は $h$ に
よらずこの恒等式を**恒等的に満たす**（同じ credit と同じ slope から作られている
から）のに対し、**モデルの真の対称性は連動時にしか存在しない**（§12.1）。つまり
**固定 $h$ では、推定量はネットワークが持たない対称性を強制している。**
これはサンプル数では消えない構造バイアスであり、**§10.4 の「$h$ 連動が推定量を
45% 改善する」を $O(h^2)$ 誤差より深いレベルで説明する。**

### 12.5 これが $c_\infty \approx 0.95$ を説明する

§11.3 では $c_\infty$ の残差を「`cov_jac` credit の構造」とだけ書いたが、§12.4 から
もっと具体的な説明が出る。**σ 勾配は重み勾配の全体ではなく、動径射影という
低次元成分だけを読む。** 真の動径成分が小さい、多数の項が相殺する、接線成分に比べて
S/N が低い、のいずれかなら、射影後の cosine は大きく落ちる。実測:

| | mean | median | max |
|---|---|---|---|
| 層 1 の $\lvert\text{radial}\rvert/(\lvert a\rvert\lvert g\rvert)$ | 0.359 | 0.259 | 0.999 |
| 層 2 | 0.204 | 0.191 | 0.358 |

**動径成分は勾配のノルムの 2〜4 割しかない。** したがって重み勾配全体の cosine が
0.998（本体 §5.3）でも、その動径射影の cosine が 0.95 に落ちることは自然である。
**§11.7 の課題 1 に対する最有力の仮説**であり、検証は次の 3 つの比較でできる:
(1) 直接計算した $g_{\log\sigma}$、(2) $-(w_k^\mathsf{T}g_{w_k}+b_k g_{b_k})$ から
再構成した値（同一 credit なら (1) と機械精度で一致するはず。§12.4 で確認済み）、
(3) 高予算ゼロ次参照（§11.3 の大 $T$ SPSA がそのまま使える）。さらに seed ごとに
$r_k = |a_k^\mathsf{T}g_{a_k}|/(\|a_k\|\|g_{a_k}\|)$ を測り、$r_k$ が小さい seed ほど
$c_\infty$ が低いかを見れば、**§11.2 の seed ばらつき（0.82〜0.97）も説明できる
可能性がある。**

### 12.6 overlap と mean_P はゲージ依存（実測）

$\mathrm{overlap} = \cos(P_A, P_B)$ も $\mathrm{mean}(P)$ も、成分ごとの
リスケールで値が変わるのでゲージ不変ではない。2 コンテキストは $a_k$ を共有するので
$\alpha_k$ は両方の場を同時に動かし、**両タスクの関数は不変のまま**:

| | overlap | mean_P | max\|dy_A\| | max\|dy_B\| |
|---|---|---|---|---|
| ゲージ前 | 0.8305 | 0.5016 | — | — |
| ゲージ後 | **0.8720** | **1.1033** | **0.00e+00** | **0.00e+00** |

同じ操作の下で、比 $\sigma_{B,k}/\sigma_{A,k}$ の変化は **1.19e-07**、すなわち
浮動小数点の丸め以内で**不変**である。

**同一の関数を計算するパラメータ設定どうしで overlap が 0.83 と 0.87、mean_P が
0.50 と 1.10 になる。** これらは関数の量ではなくゲージの量であり、それ自体では
何の証拠にもならない。帰結:

- **§10.3 の overlap・mean_P 列は使えない**（連動下の実験のため）。H3 の証拠は
  $h$ 固定の §9.3・§9.4 に限られる。損失はゲージ不変なので H1・H2 は無傷。
- **§9.5 の「σ 勾配は場を一律に縮める → 受容野が鋭くなる」という解釈も、
  連動下では成立しない。** $|a_k|$ が比例して縮めば実効的な応答幅は変わらない。
  プロットすべきは σ 単独ではなく**実効ゲイン $\gamma_k = \|[w_k,b_k]\|/\sigma_k$**
  と、**相対場 $\tilde s_{r,k} = \log\sigma_{r,k} - \frac1R\sum_{r'}\log\sigma_{r',k}$**
  （ユニットごとにコンテキスト平均を抜いた対数場＝ゲージ固定）である。
- **正則化 $\lambda_\text{sparse}\mathrm{mean}(P) + \lambda_\text{overlap}
  \cos(P_A,P_B)$ 自体がゲージ依存関数**なので、**タスク損失を一切悪化させずに
  正則化だけを下げるゲージ方向が存在する**。§6 の電力ゲーティングの主張も同じ罠に
  かかる: ノイズ電力だけを最小化すると解がゲージ方向へ逃げ、σ が縮んだ分だけ重み
  振幅が増えて、量子化範囲や乗算器ビット幅など別のハードウェアコストに移るだけになる。

### 12.7 主張への影響と、残る擁護

**否定的な帰結は正直に受け止めるべきである。** 連動下では σ は独立な自由度ではなく、
「θ と P は 2 つの独立な学習自由度」とは言えない。§1 の枠組みは §12.3 の 3 分解に
置き換わる。

**それでも続編は成立する。** 擁護は 3 つあり、強い順に:

1. **ハードウェアでは σ が安い座標である。** 数学的に冗長でも、アナログのノイズ
   振幅を変えるほうが重み行列を書き換えるより桁で安い。**「安い座標で高い座標の
   効果を得る」ことこそ NNN の工学的主張**であり、ゲージ冗長性はそれを否定するどころか
   前提である。しかも P が動かせるのは動径 1 次元だけなので、**「θ は計算、P は
   ゲート／動員」という §1 の直観は、「P の作用は θ の動径部分空間に一致する」という
   正確な形で回収できる**（「直交」ではなく「部分空間」）。
2. **shape parameter は重みで代替できない真に新しい自由度である。**
   位置 $\mu$ はバイアス $b$ と並進双対、スケール $\sigma$ は重みの動径成分と
   スケール双対だが、**歪度や裾指数はどちらでもない。** `nnn/noise.py` には
   安定分布の $\alpha,\beta,\gamma$ が既にあるので、shape field まで進める足場がある。
   ゲージ問題への最もクリーンな逃げ道。
3. **相対場はゲージ不変な実体である**（§12.3）。多重化のデモは、担い手を
   $\sigma_B/\sigma_A$ に読み替えれば残る。

**もう 1 つの選択肢は、連動をやめること。** 固定 $h$ なら σ は本物の自由度で
（§12.1）、§1 の枠組みがそのまま生きる。代償は推定量の劣化（est/num = 0.92、
§10.1）と、§12.4 が示した「推定量がモデルにない対称性を強制する」構造バイアスで
ある。**この二択（厳密な推定量 + ゲージ冗長 vs. 本物の自由度 + 偏った推定量）が、
続編の設計上の中心的な争点である。**

### 12.8 一般化: 恒等式はスケール族に限らない

（`report1.pdf` の指摘。未検証だが代数は確認した。）一般の分布パラメータ $\lambda$ に
ついて $\bar\phi(d;\lambda) = G(F(d;\lambda))$ と書けるなら、
$\partial\bar\phi/\partial\lambda = G'(F)\,\partial_\lambda F$ と
$\partial\bar\phi/\partial d = G'(F)\,p$ より

$$\frac{\partial\bar\phi}{\partial\lambda} = -v_\lambda(d)\,
\frac{\partial\bar\phi}{\partial d},\qquad
v_\lambda(d) = -\frac{\partial_\lambda F(d;\lambda)}{p(d;\lambda)}$$

が成り立つ。$v_\lambda$ は確率質量の速度場で、輸送方程式型の表現である。
スケール族なら $\partial_\sigma F = -(d/\sigma)p$ より $v_\sigma = d/\sigma$ と
なって §4 の恒等式に戻り、位置族なら $v_\mu = 1$ となる。**同一の forward credit と
同一の $\phi_T'$ で任意の分布パラメータを学習できる**という一般定理として書ける。
本文では定理として提示し、実験は scale に限定するのが焦点を保つ書き方だろう。

### 12.9 `report1.pdf` の他の指摘の評価

- **正しい**: Rademacher 摂動での下限 $d-1$（§11.4 に反映済み）。overlap 正則化が
  分化を直接報酬にしているので、低い overlap それ自体は自発的なモジュール発見の
  証拠にならないという指摘（$\lambda_\text{overlap}=0$ での分化、同一タスク 2 つでの
  対照、ユニットごとの勾配干渉 $C_k = g_{\log\sigma_{A,k}} g_{\log\sigma_{B,k}}$ の
  測定を提案している。$C_k < 0$ のユニットで場が分離し $C_k>0$ で共有が維持されると
  示せれば、overlap の値そのものよりはるかに一般性が高い）。「真の勾配」が有限

  §11.4 の次元優位は弱まるので、主張は「局所 credit から空間場を学習できる」に
  置くべき、という但し書きも正しい）。

## 付録: 恒等式の数値検証スクリプト

```python
# scale family p(xi) = (1/s) q(xi/s):  d phi_bar/d s == -(d/s) * phi_bar'(d)
# gaussian (s=0.5): max|num - pred| = 3.0e-11
# uniform  (r=1.0): max|num - pred| = 1.3e-10
import numpy as np
from scipy.stats import norm

def phibar_gauss(d, s):
    F = norm.cdf(d / s); return 2 * F * (1 - F)

def phibar_prime_gauss(d, s):
    F = norm.cdf(d / s); p = norm.pdf(d / s) / s
    return 2 * (1 - 2 * F) * p

d, s0, h = np.linspace(-0.45, 0.45, 9), 0.5, 1e-6
num  = (phibar_gauss(d, s0 + h) - phibar_gauss(d, s0 - h)) / (2 * h)
pred = -(d / s0) * phibar_prime_gauss(d, s0)
assert np.abs(num - pred).max() < 1e-9
```
